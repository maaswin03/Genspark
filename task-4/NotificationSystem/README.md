Topic : Simple Notification System

Consist of Layers :
1.Models - (User , Notification , SMSNotication , EmailNotification)
2.InterfaceLayer (IUserService , INotificationService , INotification)
3.ServiceLayer(UserService , NotificationService)

Key features : 
1.Create new user
2.Delete a user based on id
3.Fetch user based on Email
4.Fetch user based on Phone number
5.Fetch all the created users
6.Send notification (Email / SMS)
7.Unsend notification
8.Shows all the notification
