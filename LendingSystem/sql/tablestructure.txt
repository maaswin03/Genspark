Store Procedures

Borrow Book
1. Their membership is inactive
2. They already reached the borrowing limit
3. The book copy is not available
4. They have unpaid fines above ₹500
5. They already borrowed the same book and have not returned it
6. If success update bookCopies ,  insert in borrowing

Return Rules
1. The system should calculate late return fine
2. Fine amount = ₹10 per delayed day
3. The returned book copy should become available again
4. Borrowing status should change to returned
5. Return date should be saved
6. Update book copies and borrowing and insert in fines if available


Functions
1. calculate_member_fine(member_id) - for calculating fine
2. get_available_books_by_category(category_id) - for returning available books by category
3. get_member_borrowing_summary(member_id) -  for returning member borrowing summary
